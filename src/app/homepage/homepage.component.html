Introduction

Angular is an open source javascript framework, it is used to make SPAs (single page applications).
Angular is created and maintained by Google and is completely written in typescript which is a superset of javascript.
There are many versions available in angular, Angularjs is the first version of angular which is written in javascript. Learn more about angular here..

Angular versions: Angularjs->(Angular2->Angular19)->Angular20  (to be stable)

Features
  1. It is used to create SPA
  2. It has two way databinding feature
  3. It is written in typescript
  4. It has powerful command line interface (angular cli)
  5. It is component based framework
  6. It is fast in performance
