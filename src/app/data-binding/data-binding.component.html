Data binding in angular is an important feature which is used for communication between typescript component and html template.

One way databinding
Name Description
String interpolation {{ name }} this value is string interpolation
Property binding image property [src] is binding
Event binding this is event binding
Two way databinding

Hello {{ username }}

Two way binding uses [(ngModel)] directive which helps to communicate both sides at the same time.
To use [(ngModel)], FormsModule must be imported in app module.
