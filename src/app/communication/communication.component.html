In order to communicate between two components, angular provides special feature for component communcation with the help of @Input and @Output decorators. This is good only when there is a communcation between parent and child component.
Name Description
@Input
{{ input }}
@Output
This decorator is used when the communcation is from child to parent component. To emit an event from outside a component this decorator is used, it will inform angular that the emmiting value is coming from outside component, it will use EventEmitter() to create custom event and to use this decorator it must be added below the Input decorator.It is achieved by event binding.
