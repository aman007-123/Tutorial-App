Directives in angular are used to give instructions in the DOM. Components are also directives, because through components we are instructing angular to place the HTML and TypeScript logic in the place of selector. Directives are of three types.

Component directives

Components are having typescript file and html template in which business logic are typed, through this logic, component is instructing angular to place the logic in selector.

Structural directives

They are used in angular to modify the DOM. Example: *ngIf, *ngFor, *ngSwitch

Name Description
*ngIf {{ text }} This is false text
*ngFor {{ value }}
*ngSwitch Using switchcase we can switch between different DOM based elements
Attribute directives

They are used in angular to modify the property of DOM. Example: *ngClass, *ngStyle

Name Description
*ngClass It is used to add dynamically class in the DOM
*ngStyle It is used in the DOM to apply styles dynamically. It uses js objects
Custom directives

If there is different requirement, other than the predefined directives in angular, we can create our own directive by this command.
ng generate directive 'directive-name'
